How to use this example?
========================
1. Make sure that CS2PHP solution is compiled in RELEASE mode. 
   We need following file Lang.Cs2Php\bin\Release\net472\cs2php.exe
2. Run compile.bat
3. Look into _ExamplesOut\E04_Graphics

What is in example?
=============
Example shows how to create image dynamically and embed it within web page.
